# innovation

A visualization game about innovation & discipline.

## Development

This project is built with React + Vite.

### Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

### Deployment

This project is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the `main` branch.

**Live Site:** https://j143.github.io/innovation/

## Tech Stack

- [React](https://react.dev/) - UI library
- [Vite](https://vite.dev/) - Build tool and dev server
- [GitHub Pages](https://pages.github.com/) - Hosting
